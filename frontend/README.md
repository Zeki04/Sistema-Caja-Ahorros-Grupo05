# Frontend
Interfaz de usuario del Sistema de Caja de Ahorros.
Este directorio contendrá la interfaz de usuario desarrollada en React.js y Tailwind CSS.

### Stack Tecnológico
   - **Librería:** React.js
   - **Estilos:** Tailwind CSS
   - **Comunicación:** Peticiones HTTP asíncronas hacia la API RESTful.
