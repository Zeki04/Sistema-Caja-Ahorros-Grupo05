# Entorno de Pruebas Unitarias

Este directorio contiene los scripts de pruebas unitarias para el backend de la aplicación utilizando la herramienta `Pytest`.

Para ejecutar la batería de pruebas y medir el porcentaje de cobertura del código (el cual debe ser mínimo del 60%), ubíquese en la raíz del directorio backend y ejecute el siguiente comando en la terminal:

`pytest --cov=.`
